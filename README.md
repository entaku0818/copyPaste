# CopyPaste

iOSアプリケーションのコピー＆ペースト機能を管理するアプリケーションです。

## 機能

- クリップボードの履歴管理
- テキストのコピー＆ペースト
- カスタムスニペットの保存と管理

## 開発環境

- Xcode 15.0以上
- iOS 17.0以上
- Swift 5.9以上

## インストール方法

1. リポジトリをクローン
```bash
git clone https://github.com/yourusername/copyPaste.git
```

2. Xcodeでプロジェクトを開く
```bash
cd copyPaste
open copyPaste.xcodeproj
```

3. ビルドして実行
