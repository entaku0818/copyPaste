# バックグラウンドでのクリップボード監視と自動保存機能

## 機能要件
- アプリがバックグラウンドでも継続的にクリップボードを監視
- 新しいコピー操作を検知したら自動的に履歴に保存
- バックグラウンド実行中であることをユーザーに通知

## 技術要件
- バックグラウンドタスクの実装
  - UIApplicationのバックグラウンドタスク機能を使用
  - バッテリー消費を最適化
- クリップボード監視の実装
  - UIPasteboardの変更通知を使用
  - 不要な重複保存を防止

## UI要件
- ステータスバーまたは通知センターで監視状態を表示
- バックグラウンド実行のON/OFF設定
- バッテリー消費に関する警告と設定

## セキュリティ要件
- プライバシーに配慮したデータ取り扱い
- センシティブなデータの扱いに関する設定オプション
